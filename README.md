The repo contains the code for a game I made called Pattern Lock

It is a memorization game where a randomly generated sequence of LEDs and their corresponding unique sounds plays
each LED and corresponding sound both correspond to a button, for example c# and the LED on the 2nd pin correspond to the button on the 7th pin
to run the code on your machine there is a couple of things you have to do:
- first of all: you need to wire your LEDs and buttons and buzzer just as I put in the code(of course you can change them, I chose the pins that worked on my arduino)
- then you will have to install libraries for python, before I say which libraries I need to make something clear
  you can use EITHER of the gui codes, the one made in pygame is better in my opinion.
  after you decide which gui code you want to use then see the first few lines, what did it say to import ? downlad them, delete the other gui code, and start playing!
  (or modifying the code to make your own game :) )
- now you're good to go <3

this code was made with the help of AI, I used it as a companion because I wanted to finish as quickly as possible.
